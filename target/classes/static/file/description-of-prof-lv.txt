四川大学计算机学院（软件学院）院长 教授，国家杰出青年基金获得者 教育部“新世纪优秀人才”支持计划获得者，中国计算机学会青工委委员，中国计算机学会模式识别与人工智能专委会委员，中国人工智能学会机器学习专委会委员，四川省“学术和技术带头人”，四川省“有突出贡献的优秀专家”。研究方向：神经网络、机器学习与大数据分析。 师从IEEE Fellow、章毅教授研究神经网络十余年。在IEEE TNN, IEEE TFS, IEEE TSMC, IEEE TNNLS, IEEE TSP等期刊和AAAI, IJCAI, IJCNN等重要国际会议发表学术论文80余篇。出版英文专著《Subspace Learning of Neural Networks》。2011年5月被YBP（面向全球图书馆的信息服务商）评为计算机科学最畅销书之一。获得授权发明专利7项。2012年获得教育部自然科学一等奖，2011年获得四川省科技进步一等奖，2007年获CCF优秀博士论文奖。担任IEEE Trans. Neural Networks and Learning Systems编委（Associate Editor）
